Check that your system meets the requirements before installing Crimsonland.

## Python version

<Note>
Crimsonland requires **Python 3.13 or higher**.
</Note>

Check your Python version:

```bash
python3 --version
```

If you need to install or upgrade Python:

<Tabs>
  <Tab title="Linux">
    ```bash
    # Ubuntu/Debian
    sudo apt update
    sudo apt install python3.13
    
    # Fedora
    sudo dnf install python3.13
    
    # Arch
    sudo pacman -S python
    ```
  </Tab>
  
  <Tab title="macOS">
    ```bash
    # Using Homebrew
    brew install python@3.13
    ```
  </Tab>
  
  <Tab title="Windows">
    Download from [python.org](https://www.python.org/downloads/) or use:
    ```powershell
    # Using winget
    winget install Python.Python.3.13
    ```
  </Tab>
</Tabs>

## Platform-specific requirements

### Linux

<Warning>
**Wayland users**: Current PyPI raylib wheels are X11-oriented on x86_64. You may need `xwayland` and `libX11`.

See [electronstudio/raylib-python-cffi#199](https://github.com/electronstudio/raylib-python-cffi/pull/199) for details.
</Warning>

Install X11 dependencies if needed:

```bash
# Ubuntu/Debian
sudo apt install xwayland libx11-6

# Fedora
sudo dnf install xorg-x11-server-Xwayland libX11

# Arch
sudo pacman -S xorg-xwayland libx11
```

### macOS

No additional dependencies required beyond Python 3.13+.

### Windows

No additional dependencies required beyond Python 3.13+.

## Hardware requirements

### Minimum

- **CPU**: Any modern dual-core processor
- **RAM**: 512 MB
- **GPU**: OpenGL 3.3 compatible graphics card
- **Storage**: 100 MB free space

### Recommended

- **CPU**: Quad-core processor or better
- **RAM**: 2 GB
- **GPU**: Dedicated graphics card with OpenGL 4.3+
- **Storage**: 500 MB free space (for assets and replays)
