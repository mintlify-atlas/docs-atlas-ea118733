# Perks System

Perks are passive abilities chosen on level-up. The player picks one from a random selection of 5 (or more with Perk Expert/Master). Most perks are permanent, some are instant one-shots, and a few carry harsh trade-offs.

## Perk Selection Mechanics

### Level-Up Process

1. Gain experience from kills and bonuses
2. Reach XP threshold to level up
3. Game pauses and displays perk selection screen
4. Choose 1 perk from offered selection
5. Perk applies immediately
6. Resume gameplay

### Selection Pool Size

**Base**: 5 perks offered per level

**With Perk Expert** (perk 21): 7 perks offered

**With Perk Master** (perk 43): 7 perks offered (NOTE: Description says doubles, but runtime shows +2 like Expert)

From `src/crimson/perks/ids.py:10-16`:

```python
class PerkFlags(IntFlag):
    QUEST_MODE_ALLOWED = 0x1
    TWO_PLAYER_ALLOWED = 0x2  
    STACKABLE = 0x4
```

### Availability Filters

**Quest Mode**: Only perks with `QUEST_MODE_ALLOWED` flag

**Two-Player**: Only perks with `TWO_PLAYER_ALLOWED` flag

**Stackable**: Can appear even if already owned

### Two-Player Perk Sharing

In co-op, perks are shared between players. Picking a perk grants it to all players simultaneously.

**Description**: "There comes a time in each man's life when you'd just rather not move anymore. Being a living fortress comes with extra benefits as well. You do more damage the longer you stand still."

**Effect**: 
- While standing still: timer ramps up over 30 seconds
- Bullet damage multiplier: `timer × 0.05 + 1`
- Max: ×2.5 at 30 seconds
- Moving resets timer
- In multiplayer: each player's timer stacks

**Applies To**: Damage type 1 (bullets)

**Availability**: All modes
**Stackable**: No
**Prerequisites**: None
</Accordion>

**Damage taken while reloading ×0.5**

**Description**: "Damage received during reloading a weapon is halved."

**Effect**: All damage ×0.5 while reload timer active

**Availability**: All modes
**Stackable**: No
**Prerequisites**: None
</Accordion>

**Availability**: All modes
